# Myntra Functional Clone

A feature-rich, multi-page e-commerce landing page clone of the Myntra platform. This project goes beyond UI by implementing core shopping cart logic using **Vanilla JavaScript**.

## 🎯 Key Functionalities
* **Product Catalog:** Interactive display of items with dynamic styling.
* **Add to Bag:** Real-time updates to the bag count using JavaScript.
* **Shopping Bag Page:** A dedicated view to review selected items.
* **Dynamic Bag Summary:** Automated calculation of Total MRP, Discounts, and Final Amount.
* **Remove from Bag:** Seamlessly delete items from the cart with instant total recalculation.
* **Persistent State:** (Optional tip: mention if you used LocalStorage to save the cart).

## ✨ UI Features
* **Navigation Bar:** Functional search and profile icons with dropdowns.
* **Responsive Layout:** Built with CSS Flexbox and Grid for all screen sizes.
* **Hover Effects:** Smooth transitions on product cards and buttons.

## 🛠️ Technologies Used
* **HTML5:** Semantic markup for e-commerce structure.
* **CSS3:** Custom styling and responsive media queries.
* **JavaScript (ES6):** Logic for cart management, DOM manipulation, and data handling.
